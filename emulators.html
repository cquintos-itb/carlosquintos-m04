DeSmuME

DeSmuME is the most seasoned open source Nintendo DS emulator. It sports tools designed for hackers, speedrunners, youtubers, and casual gamers. Homebrew programs are supported and can even be debugged through GDB.

Downloads

  • Windows (64-bit)
  • macOS

Emuladores para Android

My OldBoy!

My OldBoy! is a Gameboy Color (GBC) emulator that runs on Android platform. My OldBoy! 1.3.5 was developed by Fast Emulator and you can run Gameboy Color (GBC) games with it. The emulator's size is 1.4MB and you can download it by clicking the button below. Have in mind that the emulator does not include game roms.
